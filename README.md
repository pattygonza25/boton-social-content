# boton-social-content

Este repositorio contiene los diseños HTML de contenido social de Bot-On, organizados por semana.

Cada archivo HTML es una pieza visual independiente que se captura como imagen PNG para publicar en Instagram o LinkedIn.

## Estructura

Los archivos siguen la convención de nombres: `semanaX-plataforma-audiencia.html`

- `semana1-linkedin-nora.html` — LinkedIn 1200×628px, audiencia general (lanzamiento Venezuela)
- `semana1-ig-adultomayor-1.html` — Instagram 1080×1080px, adulto mayor (memoria)
- `semana1-linkedin-teo.html` — LinkedIn 1200×628px, adulto joven (pico cognitivo)
- `semana1-ig-hijocuidador.html` — Instagram 1080×1080px, hijo cuidador (señales de alerta)
- `semana1-ig-adultomayor-2.html` — Instagram 1080×1080px, adulto mayor (concentración)

## Recursos

Los logos de Bot-On utilizados en los diseños son:

- `Recurso_7.png` — Logo blanco (para fondos oscuros)
- `Recurso_8.png` — Logo azul (para fondos claros)

| Token | Valor |

## Captura como PNG

Abrir el archivo HTML en un navegador y capturar el div `.canvas` con las dimensiones exactas especificadas (sin scroll, sin chrome del navegador). Herramientas recomendadas: Puppeteer, playwright, o extensión de captura de pantalla a medida exacta.
